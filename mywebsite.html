Verilog

Advanced Combination Lock

This is an interactive and advanced combination lock system that I built using a microcontroller simulator called Wokwi. It allows the user to input a custom PIN, with a security feature that locks them out after three incorrect attempts. To bring the system to life, I used a servo motor to represent the lock opening, along with LEDs, green indicating access granted and red showing an incorrect PIN. A buzzer was also added for feedback, and an LED display shows what the user is typing in real time as well as whether access has been granted. This was one of my most entertaining projects, I genuinely enjoyed every second of it. I did face a few roadblocks, especially with pin connections, but it taught me patience and the importance of properly debugging both my code and wiring instead of rushing through the process. wait for my next project with a microcontroller!
